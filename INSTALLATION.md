# Installation and Uninstallation

Only download LeeHollow RustDesk Client builds from this repository's GitHub Releases page:

https://github.com/Crosis47/rustdesk/releases

## Installation

Windows installer builds should be installed by following the installer prompts. The installer or operating system may request permission before making system changes.

Portable builds, if published, can be run without installation by launching the downloaded executable.

Do not install or run LeeHollow RustDesk Client unless you understand who will provide remote support and you consent to the support session.

## Uninstallation

For installed Windows builds, use Windows Settings > Apps > Installed apps, select LeeHollow RustDesk Client or RustDesk, and choose Uninstall.

For portable builds, close the application and delete the downloaded executable or extracted folder.

Some local settings or logs may remain after uninstalling, depending on the RustDesk build and how it was used. Remove local configuration only if you are sure you no longer need saved settings or device identity information.
