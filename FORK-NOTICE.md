# Fork Notice

LeeHollow RustDesk Client is a downstream AGPLv3 fork of RustDesk.

The purpose of this fork is to produce transparent, preconfigured client builds for a self-hosted RustDesk deployment. The intended changes are limited to build-time configuration of the default ID server, relay server, and public key.

This fork does not add advertising, analytics, hidden telemetry, or unrelated remote-control functionality.

This project is not affiliated with, sponsored by, or endorsed by the upstream RustDesk project.
